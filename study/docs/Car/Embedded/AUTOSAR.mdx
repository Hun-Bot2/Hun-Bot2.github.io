# AUTOSAR (AUTomotive Open System ARchitecture)
이 파트는 선행지식이 제법 필요하기도 하고, 우선, 임베디드를 학습하려고 하는 사용자에게는 추천하지 않는다고 한다.

임베디드와 컴퓨터 구조, OS쪽을 다 알아두고, C나 C++을 자유롭게 쓰고 나서 자동차쪽으로 깊게 들어가고 싶을 때 AUTOSAR를 공부하는 것을 추천한다고 한다.
