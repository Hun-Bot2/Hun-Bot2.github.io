# Prompt Engineering

## Zero-shot Learning
Basically using LLM without any examples.
If, i have to review sentiment of the sentense "I love programming.", I can just give the prompt below.

```text
Prompt : Please classify the sentiment of the following sentence as 'positive', 'negative', or 'neutral'.
The sentence is: "I love programming."
Answer :
```

## Few-shot Learning
```text
Prompt : Pailindrome is a word, phrase, number, or other sequence of characters which reads the same backward as forward. 
Examples of palindromes include "madam", "racecar"
Examples of non-palindromes include "hello", "world"
Answer 'yes' or 'no' please.
The word 'steve' is a palindrome?

```

현 시점으로 LLM들에는 Thinking이라는 능력이 탑재되어 있고, 우리가 단계별로 생각해보세요!라고 유도하지 않아도 "Instruncion-tuning"되었기에 무언가 입력했을 때, 생각하는 과정을 거쳐 답변을 도출해낸다.
