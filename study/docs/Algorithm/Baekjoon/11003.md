## 1. 문제 분석
**접근 방식:** 아이디어 요약

**시간 복잡도:** $O(N)$

## 2. Python 풀이
테스트케이스는 맞았지만, 당연히 시간 초과

**시간초과**
```python
import sys
input = sys.stdin.readline

n,l = map(int, input().split())
A = list(map(int, input().split()))

D=[0]*n
for i in range(n):
    start_index= max(0,i-l+1)
    end_index=i
    D[i]=min(A[start_index:end_index+1])

for d in D:
    print(d, end=' ')
```
이 문제를 해결하려면 **"deque"**을 알아야 함
