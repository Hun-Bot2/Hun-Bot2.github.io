## 학습 방법론: Spaced Repetition

에빙하우스의 망각 곡선 이론에 기반하여, 한 번 푼 문제는 아래의 주기에 맞춰 총 5번 복습을 진행합니다.

* 복습 주기: 1일, 3일, 7일, 14일, 30일 후 자동 알림
* 자동화 시스템: GitHub Actions가 최초 커밋 날짜를 추적하여 매일 오전 Slack으로 복습 리스트를 전송합니다.
